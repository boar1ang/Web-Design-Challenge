Welcome to the dashboard for our weather analysis project.

The purpose of our project was to analyze how weather changes as one moves closer to the equator. Using the OpenWeatherMap API we obtained a dataset on over 500 world cities. After cleaning and organizing the data we used Pandas and Matplotlib to plot various aspects of the weather vs. latitude. Among the factors we considered were temperature, cloudiness, wind speed, and humidity.

This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed. Please contact us if you have any questions about our analysis.
